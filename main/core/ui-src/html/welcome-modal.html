Welcome to NZBHydra 2

This seems to be the first time that you started NZBHydra 2.

If you're already using NZBHydra 1 (python based) you can migrate your data.

If you're a new user (or don't want to migrate your data right now) you can start by configuring NZBHydra 2.
You will not be able to use it until you've added at least one indexer.

If you're stuck you can refer to the wiki or the online help (available from the config).
